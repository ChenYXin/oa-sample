# OA - 办公系统

## 办公自动化OA系统

- 办公自动化系统（Office Automation）是替代传统办公的解决方案
- OA系统是利用软件技术构建的单位内部办公平台，用于辅助办公
- 利用OA系统可将办公数据数字化，可极大提高办公流程执行效率

## 框架&组件

- MySQL 8.3.0
- Mybatis 3.5.16
- Alibaba Druid 1.2.23
- Servlet 6.1.0
- Vue 3.x
- Element Plus
